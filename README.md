# Reverbic

Cliente de radio por internet que funciona directamente en la terminal. Sin interfaz gráfica, sin publicidades, sin distracciones.

---

## Estaciones disponibles

- Tomorrowland One World Radio
- Tomorrowland Anthems
- Tomorrowland Daybreak Sessions
- OnlyHit Japan

---

## Requisitos

- Windows 10 o Windows 11 (64 bits)
- Cualquier terminal: Windows Terminal, PowerShell o CMD
- Conexión a internet

---

## Controles

| Tecla | Acción |
|---|---|
| `↑` / `↓` o `k` / `j` | Moverse por la lista |
| `Enter` | Reproducir la estación seleccionada |
| `Espacio` | Pausar / reanudar |
| `+` | Subir volumen |
| `-` | Bajar volumen |
| `Tab` | Cambiar el foco entre estaciones y canciones recientes |
| `p` | Escuchar 30 segundos de la canción seleccionada (via Deezer) |
| `Enter` (en canciones) | Guardar la canción en favoritos |
| `s` | Detener la reproducción |
| `q` | Salir |
